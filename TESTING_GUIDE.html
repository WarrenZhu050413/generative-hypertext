üí¨ Step 3: Test Chat Interface

Once FloatingChat appears, try these interactions:

  • Type a message like "What is this?" or "Summarize this"
  • Press Enter to send
  • Watch for the streaming response (typing indicator, then text appearing)
  • Send multiple messages to build a conversation
  • Try Shift + Enter for newlines
  • Press ESC to close the chat
üí° Note:

üîç Step 5: Test Search & Filter

Search Functionality

  • Type in the search box at the top
  • Watch cards filter in real-time (300ms debounce)
  • Search works across title, content, domain, and tags
  • Click the X to clear search

Filter Options

  1. Click the filter button (funnel icon)
  2. Try toggling "Starred only"
  3. Select domains from the dropdown
  4. Select tags (if any cards have tags)
  5. Try different date ranges
  6. Observe active filter chips at the top of the panel
  7. Click X on chips to remove individual filters
  8. Click "Clear All" to reset filters

Keyboard Shortcuts

  • Press Cmd/Ctrl + K to focus search
  • Press Cmd/Ctrl + F to toggle filter panel
  • Press ? to show keyboard shortcuts help
  • Press F to fit all cards in view
