How Focus Summary is Collected

Location: hypertext-experience.js:1225, 1333

Priority Chain:

  1. First Priority: session.lastResult.pillText
    The LLM-generated pill text from the hypertext response
    Example: "Gradient Reference", "ML Concept", etc.
  2. Second Priority: session.subject
    The original text selected by the user
    Example: "gradient descent", "machine learning", etc.
  3. Fallback: 'Active selection'
    Used if neither pillText nor subject available

Code Snippet:

const focusLabel = session.lastResult?.pillText || session.subject || 'Active selection';

Used In:

  • buildChatPayload() - Sent to chat page as focus field
  • renderConversation() - Displayed in chat tooltip meta area

Markdown Rendering Function

function renderMarkdown(text) {
  if (typeof window.marked === 'undefined') {
    return text; // Fallback to plain text
  }

  try {
    // Configure marked with safe options
    marked.setOptions({
      breaks: true,
      gfm: true,
      headerIds: false,
      mangle: false,
      sanitize: false // We'll sanitize afterward if needed
    });

    return marked.parse(text);
  } catch (error) {
    console.warn('[Hypertext] Markdown parsing failed:', error);
    return text;
  }
}
