‚ö†Ô∏è Potential Issues & Mitigations ‚ñ∂

Potential Issue: State Not Updating

  • Problem: Messages added but component doesn't re-render
  • Cause: WindowManager state mutation instead of replacement
  • Solution: Ensure immutable updates in windowManager
  • Mitigation: Use spread operator for state updates

Potential Issue: Flex Layout Breaks

  • Problem: Chat doesn't expand to fill space
  • Cause: Conflicting flex properties or min-height issues
  • Solution: Add min-height: 0 to flex children
  • Mitigation: Test with various window sizes

Common Pitfalls

  1. Nested flex containers: Need explicit min-height: 0 for overflow
  2. Message array mutations: Always create new array for state updates
  3. Streaming state cleanup: Clear streaming content after message completes
  4. Scroll position: Use scrollIntoView({ behavior: 'smooth' }) after render
