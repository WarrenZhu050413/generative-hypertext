üéì How the Modes Work

Educational Mode

When active, ChatGPT:

  1. Provides pedagogical explanations (simple ‚Üí complex)
  2. Identifies 3-5 key concepts worth exploring
  3. Calls create_hypertext() for each concept
  4. Presents links as "Explore deeper" section

Result: Every response becomes a launchpad for deeper learning

Exploratory Mode

When active, ChatGPT:

  1. Provides comprehensive topic overview
  2. Identifies 5-10 branching research directions
  3. Organizes by category (fundamentals, methods, applications, open questions)
  4. Creates extensive hypertext knowledge graph

Result: Systematic research tool with multiple investigation paths
