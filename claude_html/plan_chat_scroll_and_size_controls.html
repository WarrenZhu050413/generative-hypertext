⚠️ Potential Issues & Mitigations ▶

Edge Cases

  • Issue: Users with stored 'large' size will need migration
    Mitigation: In readStoredTextSize(), map 'large' → 'large' (no change needed)
  • Issue: Scrolling to bottom during streaming may cause jitter
    Mitigation: Keep scroll at bottom only when user hasn't manually scrolled up
  • Issue: Button layout may wrap on narrow tooltips
    Mitigation: Use compact button labels and ensure min-width is sufficient
  • Issue: Rapidly clicking +/- may cause layout shifts
    Mitigation: Debounce size changes or use CSS transitions

Accessibility Concerns

  • Ensure buttons have proper aria-label attributes
  • Announce size changes to screen readers
  • Maintain keyboard navigation between buttons
