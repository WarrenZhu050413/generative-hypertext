üìã InlineChatWindow Image Integration with Shared Architecture

Objective

Enable InlineChatWindow to accept drag-and-drop images using the existing shared architecture, then send them to Claude's Vision API for multimodal conversations.

Complexity: Low-Medium
Reuses: ImageUploadZone + imageService

Architecture Strategy

  1. Reuse ImageUploadZone component
  2. Reuse imageService for processing
  3. Extend Message type for images
  4. Transform messages for Vision API
  5. Don't create cards (different from Canvas/SidePanel)

    File: src/components/InlineChatWindow.tsx:76-129
    const handleSendMessage = async () => {
  if ((!inputValue.trim() && pendingImages.length === 0) || isStreaming) {
    return;
  }

  const userMessage: Message = {
    role: 'user',
    content: inputValue.trim() || '(Image attached)',
    images: pendingImages.length > 0 ? [...pendingImages] : undefined
  };

  setInputValue('');
  setPendingImages([]); // Clear pending images

  const newMessages = [...messages, userMessage];
  setMessages(newMessages);
  setIsStreaming(true);

  try {
    const hasImages = userMessage.images && userMessage.images.length > 0;

    if (hasImages) {
      // Transform to Claude's multimodal format
      const { claudeAPIService } = await import('@/services/claudeAPIService');

      const claudeMessages = newMessages.map(m => ({
        role: m.role,
        content: m.images && m.images.length > 0
          ? [
              // Images first
              ...m.images.map(img => {
                const mediaType = img.dataURL.match(/data:([^;]+);/)?.[1] || 'image/png';
                const base64Data = img.dataURL.split(',')[1];
                return {
                  type: 'image',
                  source: {
                    type: 'base64',
                    media_type: mediaType,
                    data: base64Data
                  }
                };
              }),
              // Text last
              { type: 'text', text: m.content }
            ]
          : m.content // Text-only message
      }));

      const systemPrompt = isElement
        ? formatElementContextAsPrompt(initialContext)
        : formatPageContextAsPrompt(initialContext);

      const response = await claudeAPIService.sendMessage(
        claudeMessages,
        { system: systemPrompt, maxTokens: 4096 }
      );

      setMessages([...newMessages, { role: 'assistant', content: response }]);
    } else {
      // Text-only - use streaming API (existing code)
      const { chatWithPage } = await import('@/services/claudeAPIService');
      const systemPrompt = isElement
        ? formatElementContextAsPrompt(initialContext)
        : formatPageContextAsPrompt(initialContext);

      let assistantContent = '';
      const stream = await chatWithPage(systemPrompt, newMessages.map(m => ({
        role: m.role,
        content: m.content
      })));

      for await (const chunk of stream) {
        assistantContent += chunk;
        setStreamingContent(assistantContent);
      }

      setMessages([...newMessages, { role: 'assistant', content: assistantContent }]);
      setStreamingContent('');
    }
  } catch (error) {
    console.error('[InlineChatWindow] Error:', error);
    setMessages([...newMessages, {
      role: 'assistant',
      content: '‚ùå Error: Could not connect to API. Please check your backend is running or API key is configured.'
    }]);
  } finally {
    setIsStreaming(false);
  }
};

Why This Architecture Works

  • DRY Principle: Same code path as Canvas/SidePanel for image processing
  • Consistency: Same validation logic, same error messages
  • Tested: imageService has 100% unit test coverage (21 tests)
  • Maintainability: Bug fixes in shared code benefit all consumers
üîç Key Differences: InlineChatWindow vs Canvas/SidePanel ‚ñ∂
Aspect Canvas/SidePanel InlineChatWindow (NEW)
Goal Save images as Card objects Attach images to chat messages
Storage Immediately saved to chrome.storage.local Temporarily held in React state (pendingImages)
Persistence Permanent (until user deletes card) Ephemeral (cleared after sending message)
API Call No API call - just storage Sends to Claude Vision API
Message Format N/A Multimodal content blocks (image + text)
Use Case Capture and organize web content Ask questions about images
