## Resize & Drag Systems

Nabokov uses two different systems for drag & resize:

### react-rnd (FloatingWindow, InlineChatWindow)
- **Context:** Floating windows outside of React Flow canvas
- **Coordinates:** Absolute positioning (px from viewport)
- **Features:** Custom resize handles, bounds checking, z-index management
- **Why:** Purpose-built for floating UI elements

### NodeResizer (CardNode)
- **Context:** Cards on React Flow canvas
- **Coordinates:** React Flow graph coordinates (transformed by zoom/pan)
- **Features:** Zoom-aware, snapping, alignment guides, multi-select
- **Why:** Designed for React Flow's coordinate system

This separation is intentional and architecturally appropriate.
