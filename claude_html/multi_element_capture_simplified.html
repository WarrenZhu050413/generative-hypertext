Simplified Concept: Combine All Into One Card

User Flow: Press hotkey ‚Üí Cmd/Ctrl-click multiple elements ‚Üí Click "Capture" ‚Üí All elements merged into single card

  • Selection Mode: Hold Cmd/Ctrl while clicking to add elements to selection
  • Visual Feedback: Each selected element gets numbered badge + red outline
  • Action Panel: Shows count and "Capture" + "Clear" buttons
  • Result: Single card with all element HTML combined in selection order

2. First Click User Cmd/Ctrl-clicks first element
‚Üí Red outline + badge "1" appears

3. Additional Selections User Cmd/Ctrl-clicks more elements
‚Üí Each gets numbered badge (2, 3, 4...)
‚Üí Panel updates: "3 elements selected"
‚Üí Click selected element again to deselect
4. Capture User clicks "Capture" in action panel

‚Üí Single card created (Canvas or Stash)

SelectedElement Interface

interface SelectedElement {
  element: HTMLElement;
  index: number;              // Selection order (1, 2, 3...)
  selector: string;           // CSS selector for display
  html: string;               // Sanitized outerHTML (captured immediately)
  position: { x: number; y: number };
  dimensions: { width: number; height: number };
}

Toggle Selection Logic

function toggleSelection(el: HTMLElement) {
  const idx = selections.findIndex(
    s => s.element === el
  );

  if (idx >= 0) {
    // Deselect: remove and renumber
    removeSelection(idx);
  } else {
    // Add: capture HTML immediately
    addSelection(el);
  }
}

Combine & Create Card

function combineAndCapture() {
  // Combine all HTML in selection order
  const combinedHTML = selections
    .map((sel, idx) => `
      <div class="captured-element" data-index="${idx + 1}">
        ${sel.html}
      </div>
    `)
    .join('\n\n');

  // Create single card
  const card: Card = {
    id: generateId(),
    content: sanitizeHTML(combinedHTML),
    metadata: {
      url: window.location.href,
      title: document.title,
      domain: window.location.hostname,
      // ... other metadata
    },
    cardType: 'clipped',
    stashed: mode === 'stash',
    tags: [],
    starred: false,
    createdAt: Date.now(),
    updatedAt: Date.now()
  };

  // Save to storage
  await saveCard(card);

  // Success feedback
  showToast(`Combined ${selections.length} elements into one card`, 'success');

  // Clear selections
  clearSelections();
}

test('should select multiple elements and combine into single card', async ({ page }) => {
  // Navigate to test page
  await page.goto('http://example.com');

  // Activate selector
  await page.evaluate(() => {
    chrome.runtime.sendMessage({ 
      type: 'ACTIVATE_SELECTOR', 
      mode: 'canvas' 
    });
  });

  // Select 3 elements with Cmd+Click
  await page.click('.element-1', { modifiers: ['Meta'] });
  await page.click('.element-2', { modifiers: ['Meta'] });
  await page.click('.element-3', { modifiers: ['Meta'] });

  // Verify badges
  await expect(page.locator('.selection-badge:has-text("1")')).toBeVisible();
  await expect(page.locator('.selection-badge:has-text("2")')).toBeVisible();
  await expect(page.locator('.selection-badge:has-text("3")')).toBeVisible();

  // Verify action panel
  await expect(page.locator('.multi-select-panel')).toBeVisible();
  await expect(page.locator('.count')).toHaveText('3 elements selected');

  // Capture
  await page.click('.capture-btn');

  // Verify single card created
  const cards = await page.evaluate(() => {
    return chrome.storage.local.get('cards');

  expect(cards.cards).toHaveLength(1);
  expect(cards.cards[0].content).toContain('data-index="1"');
  expect(cards.cards[0].content).toContain('data-index="2"');
  expect(cards.cards[0].content).toContain('data-index="3"');
});

‚ö†Ô∏è Edge Cases

Element Limit

Problem: User selects 50+ elements

Solution: Max 20 elements, toast: "Maximum 20 elements. Please capture current selection first."

Nested Elements

Problem: User selects parent, then child

Solution: Allow but show warning badge: "‚ö†Ô∏è Element 3 is inside Element 1"

Dynamic Content

Problem: Element changes after selection

Solution: Capture HTML immediately on selection, not on capture button click

Large Combined Card

Problem: 10 elements = 500KB HTML

Solution: Show warning toast: "Combined content is large (500KB). Consider fewer elements."
