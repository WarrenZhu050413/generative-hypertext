🚀 Element-Attached Chat Enhancements

📋 Overview: Five major enhancements to element-attached chat system for improved usability, flexibility, and integration with existing canvas/stash workflows.

🎯 Feature Summary

1. Save Conversation to Stash/Canvas HIGH

Goal: Allow users to save element + chat conversation as a card to either stash or canvas

  • Add "Save to Stash" and "Save to Canvas" buttons to ElementChatWindow header
  • Create card with element HTML content + full conversation history
  • Support both stashed and canvas workflows
  • Maintain conversation metadata and context

2. Input While Generating LOW

Goal: Enable typing in input box even during AI response streaming

  • Remove disabled={isStreaming} from textarea
  • Allow user to compose next message while waiting
  • Improve perceived responsiveness

3. Message Queueing MEDIUM

Goal: Queue messages sent during streaming instead of blocking

  • Maintain queue of pending user messages
  • Process queue automatically when streaming completes
  • Show visual indicator of queued messages
  • Support queue cancellation

4. Image Upload Support MEDIUM

Goal: Add drag-drop image support to element chat (like InlineChatWindow)

  • Integrate ImageUploadZone component
  • Support multimodal API calls with images
  • Show image previews in chat
  • Include images in saved conversations

5. Chat with Highlighted Text MEDIUM

Goal: Select text on page and press Ctrl+Shift+C to open chat about that text

  • Detect text selection when activating chat selector
  • Create contextual chat with selected text as focus
  • Pre-populate system prompt or first message
  • Show selected text in chat UI for context
▶Feature 1: Save Conversation to Stash/Canvas - Detailed Plan

Architecture Overview

Data Flow

  1. User clicks "Save to Stash" or "Save to Canvas" in ElementChatWindow
  2. System captures: element HTML, element metadata, full conversation, timestamps
  3. Creates Card object with cardType: 'element-with-chat'
  4. Stores card in chrome.storage.local (stashed or on canvas)
  5. Broadcasts update event for UI refresh

export async function convertElementChatToCard(
  session: ElementChatSession,
  elementDescriptor: ElementDescriptor,
  stashImmediately: boolean
): Promise<Card> {
  // Get element HTML
  const element = document.querySelector(`[data-nabokov-chat-id="${session.elementId}"]`);
  const elementHTML = element?.outerHTML || '';

  // Format conversation as markdown
  const conversationMarkdown = formatConversationAsMarkdown(session.messages);

  // Create card
  const card: Card = {
    id: generateId(),
    content: `
      <div class="element-with-chat">
        <div class="original-element">${elementHTML}</div>
        <div class="chat-transcript">
          <h3>💬 Conversation</h3>
          ${conversationMarkdown}
        </div>
      </div>
    `,
    metadata: {
      url: session.pageUrl,
      title: document.title,
      domain: new URL(session.pageUrl).hostname,
      // ... other metadata
    },
    cardType: 'element-with-chat',
    stashed: stashImmediately,
    conversation: session.messages, // Preserve original chat
    tags: ['element-chat'],
    createdAt: Date.now(),
    updatedAt: Date.now(),
    starred: false
  };

  return card;
}

const handleSendMessage = async () => {
  if (!inputValue.trim()) return;

  const userMessage = inputValue.trim();
  setInputValue('');

  // If currently streaming, add to queue
  if (isStreaming) {
    console.log('[ElementChatWindow] Queueing message:', userMessage);
    setMessageQueue(prev => [...prev, userMessage]);
    return; // Don't send immediately
  }

  // Otherwise send normally
  await sendMessageToAPI(userMessage);
};

const sendMessageToAPI = async (userMessage: string) => {
  try {
    setIsStreaming(true);

    // ... existing API call logic ...

  } catch (error) {
    // ... error handling ...
  } finally {
    setIsStreaming(false);

    // Process queue if any messages waiting
    if (messageQueue.length > 0 && !isProcessingQueue) {
      processQueue();
    }
  }
};

const processQueue = async () => {
  if (messageQueue.length === 0 || isProcessingQueue) return;

  setIsProcessingQueue(true);

  // Get next message from queue
  const nextMessage = messageQueue[0];
  setMessageQueue(prev => prev.slice(1));

  // Send it
  await sendMessageToAPI(nextMessage);

  setIsProcessingQueue(false);
};

▶Feature 5: Chat with Highlighted Text - Detailed Plan

User Flow

  1. User highlights text on page
  2. User presses Ctrl+Shift+C (chat hotkey)
  3. System detects selection and opens element chat
  4. Chat window opens with selected text as context
  5. First message pre-populated or system prompt enhanced

Implementation Steps

Step 5.1: Detect Selection on Hotkey

File: src/content/index.tsx

// In ACTIVATE_CHAT_SELECTOR handler
chrome.runtime.onMessage.addListener((message) => {
  if (message.type === 'ACTIVATE_CHAT_SELECTOR') {

    // Check for text selection
    const selection = window.getSelection();
    const selectedText = selection?.toString().trim() || '';

    if (selectedText && selectedText.length > 0) {
      console.log('[content] Text selected, opening contextual chat:', selectedText);

      // Store selection context
      const selectionContext = {
        text: selectedText,
        element: selection?.anchorNode?.parentElement,
        pageContext: {
          url: window.location.href,
          title: document.title
        }
      };

      // Open chat with text context instead of element selector
      openTextContextChat(selectionContext);

    } else {
      // No selection, proceed with normal element selector
      activateChatSelector();
    }
  }
});
Step 5.2: Create Text Context Chat Function
async function openTextContextChat(selectionContext: {
  text: string;
  element: HTMLElement | null | undefined;
  pageContext: { url: string; title: string };
}) {
  const { text, element, pageContext } = selectionContext;

  // Create a synthetic element descriptor for the selected text
  const elementDescriptor: ElementDescriptor = {
    tagName: element?.tagName.toLowerCase() || 'selection',
    id: element?.id || '',
    classes: element ? Array.from(element.classList) : [],
    textPreview: text,
    attributes: {},
    xpath: '' // Could generate XPath to selection anchor
  };

  // Generate unique ID for this text selection chat
  const chatId = `text-selection-${Date.now()}`;

  // Check if there's an existing session for this text
  const existingSession = null; // Text selections are ephemeral

  // Open element chat window with text context
  openElementChatWindow(chatId, elementDescriptor, existingSession, {
    isTextSelection: true,
    selectedText: text
  });
}
