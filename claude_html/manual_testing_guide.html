üêõ Troubleshooting

Problem: Chat window doesn't appear

Solutions:
  • Make sure you reloaded the extension after building
  • Try refreshing the web page you're testing on
  • Check Chrome DevTools console (F12) for errors
  • Try the keyboard shortcut again (Ctrl+Shift+C)

Problem: API errors when sending messages

Expected if backend not running!
  • Start the backend: npm run backend
  • Backend should show: üöÄ Nabokov Backend Server - Port 3100
  • Try sending a message again

Problem: Messages appear but no streaming

Check backend connection:
  • Look for console message: [ClaudeAPI] ‚úì Streaming via backend
  • If you see "Backend unavailable", the server isn't running
  • If you see "Direct API streaming", you need to configure an API key
