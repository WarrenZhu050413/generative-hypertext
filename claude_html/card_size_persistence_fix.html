Why It Fails

üîë Key Issue: React Flow v12's NodeResizer stores dimensions in node.width and node.height (or node.measured), NOT in node.style.width/height

The current code only checks node.style, which either:

  • ‚ùå Doesn't exist (undefined)
  • ‚ùå Contains string values like "320px" (fails the typeof === 'number' check)
  • ‚ùå Is set initially but not updated by NodeResizer

Result: The code always falls back to the default 320√ó240 dimensions.

‚úÖ Solution

Multi-Source Dimension Detection

Check multiple properties in order of preference to find the actual resized dimensions:

Step 1: Check node.width and node.height (direct properties, set by NodeResizer)
Step 2: Check node.measured.width and node.measured.height (measured dimensions)
Step 3: Check node.style.width and node.style.height (handle both numbers and strings like "320px")
Step 4: Fall back to card.size (existing saved size) or default 320√ó240

Fixed Code

// ‚úÖ FIXED CODE
const updatedCards = currentCards.map(card => {
  const node = nodesToSave.find(n => n.id === card.id);
  if (node) {
    // Helper function to extract numeric dimension from various formats
    const getNumericDimension = (value: any, fallback: number): number => {
      if (typeof value === 'number') return value;
      if (typeof value === 'string') {
        const parsed = parseFloat(value);
        if (!isNaN(parsed)) return parsed;
      }
      return fallback;
    };

    // Try multiple sources for width/height in order of preference
    const width =
      getNumericDimension(node.width, 0) ||                    // Direct property
      getNumericDimension(node.measured?.width, 0) ||          // Measured dimension
      getNumericDimension(node.style?.width, 0) ||             // Style property
      card.size?.width ||                                       // Existing saved size
      320;                                                      // Default

    const height =
      getNumericDimension(node.height, 0) ||                   // Direct property
      getNumericDimension(node.measured?.height, 0) ||         // Measured dimension
      getNumericDimension(node.style?.height, 0) ||            // Style property
      card.size?.height ||                                      // Existing saved size
      240;                                                      // Default

    return {
      ...card,
      position: node.position,
      size: { width, height },
      updatedAt: Date.now(),
    };
  }
  return card;
});
