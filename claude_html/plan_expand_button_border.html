⚠️ Potential Issues & Mitigations ▶

Edge Cases

  • Issue: Button might overlap with chat content at minimum size
    Mitigation: Use same positioning strategy as resize grip (pointer-events: none on container, auto on button). Button extends outside the border, so no overlap.
  • Issue: Icon might not be clear enough
    Mitigation: Add descriptive title/aria-label. Consider using different icon if ⤢ is unclear (can test with users).
  • Issue: Transform translate values might not perfectly center
    Mitigation: Use translate(-50%, 50%) (negative X for left corner, positive Y for bottom). Mirror of resize grip's translate(50%, 50%).
  • Issue: Button might interfere with resize grip functionality
    Mitigation: Buttons are on opposite corners (bottom-left vs bottom-right), so no interference expected.

Accessibility Concerns

  • Must be keyboard accessible (add tabindex="0")
  • Must have clear label (aria-label and title)
  • Must have visible focus indicator (button already has outline on focus)
  • Icon should be supplemented with text in tooltip/aria-label
