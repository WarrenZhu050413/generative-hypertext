📋 Plan: Try Real API First, Fall Back on Error

Updated Objective

Always try the real Claude API first (with or without explicit API key). Only fall back to mocks if the API call actually fails. Show errors to users when API fails.

Complexity: Medium
Estimated Time: 2-3 hours

New Pattern

  • Step 1: Try real Claude API (may use env credentials)
  • Step 2: If fails, show error to user
  • Step 3: Fall back to mock for development
  • User sees actual API errors, not silent failures

🔑 Key Philosophy Change

Don't check for API key before making calls. Just try the API and handle errors gracefully. This allows authentication via environment variables, Bedrock, Vertex AI, or other methods the SDK supports.

Updated claudeAPIService Pattern

async sendMessage(...) {
  // Get API key from storage (may be null)
  const apiKey = await apiConfigService.getAPIKey();

  // Use key if available, undefined if not
  // This allows SDK to use other auth methods
  const headers = {
    'Content-Type': 'application/json',
    'anthropic-version': this.API_VERSION
  };

  // Only add x-api-key if we have one
  if (apiKey) {
    headers['x-api-key'] = apiKey;
  }

  // Try request - will fail if no valid auth
  const response = await fetch(this.API_URL, {
    method: 'POST',
    headers,
    body: JSON.stringify(request)
  });

  // Let natural errors bubble up
  if (!response.ok) {
    const errorData = await response.json().catch(() => ({}));
    throw new Error(
      `Claude API error: ${response.status} ${response.statusText}${
        errorData.error?.message ? ` - ${errorData.error.message}` : ''
      }`
    );
  }

  // ... rest of success handling
}

⚠️ Error Handling Strategy ▶

Error Types to Handle

  • 401 Unauthorized: "API key is invalid or missing"
  • 429 Rate Limit: "Rate limit exceeded. Please wait and try again."
  • 500 Server Error: "Claude API is temporarily unavailable"
  • Network Error: "Network error. Check your connection."
  • Other: Show the actual error message

User-Friendly Error Wrapper

function getErrorMessage(error: any): string {
  const message = error.message || String(error);

  if (message.includes('401')) {
    return 'API key is invalid or missing. Using mock responses.';
  } else if (message.includes('429')) {
    return 'Rate limit exceeded. Using mock responses.';
  } else if (message.includes('500') || message.includes('502') || message.includes('503')) {
    return 'Claude API temporarily unavailable. Using mock responses.';
  } else if (message.includes('fetch') || message.includes('network')) {
    return 'Network error. Using mock responses.';
  } else {
    return `API Error: ${message}. Using mock responses.`;
  }
}

async *sendMessage(
  cardId: string,
  userMessage: string,
  cardContent: string
): AsyncGenerator {
  // Add user message
  const userMsg: Message = {
    id: generateId(),
    role: 'user',
    content: userMessage,
    timestamp: Date.now()
  };
  this.addMessage(cardId, userMsg);

  // Create assistant message (streaming)
  const assistantMsg: Message = {
    id: generateId(),
    role: 'assistant',
    content: '',
    timestamp: Date.now(),
    streaming: true,
    parentId: userMsg.id
  };
  this.addMessage(cardId, assistantMsg);

  // Stream response
  const controller = new AbortController();
  this.abortControllers.set(cardId, controller);

  try {
    // TRY REAL API FIRST
    console.log('[ChatService] Trying Claude API...');
    const prompt = `Context: ${cardContent}\n\nUser: ${userMessage}`;

    try {
      const response = await claudeAPIService.sendMessage([
        { role: 'user', content: prompt }
      ], {
        system: 'You are a helpful assistant analyzing web content.',
        maxTokens: 2048
      });

      console.log('[ChatService] ✓ Claude API success');

      // Simulate streaming by yielding chunks
      const chunkSize = 10;
      for (let i = 0; i < response.length; i += chunkSize) {
        if (controller.signal.aborted) break;
        const chunk = response.slice(i, i + chunkSize);
        assistantMsg.content += chunk;
        yield chunk;
        await new Promise(resolve => setTimeout(resolve, 20));
      }

    } catch (apiError) {
      // API FAILED - show error and fall back to mock
      console.error('[ChatService] ✗ Claude API failed:', apiError);
      console.warn('[ChatService] Falling back to mock');

      // TODO: Show toast to user
      // showToast(`API Error: ${apiError.message}. Using fallback.`, 'error');

      // Fall back to mock
      const stream = mockContentGenerator.generate(
        userMessage,
        cardContent,
        controller.signal
      );

      for await (const chunk of stream) {
        assistantMsg.content += chunk;
        yield chunk;
      }
    }

    assistantMsg.streaming = false;
    await this.saveConversation(cardId);

  } catch (error) {
    if (controller.signal.aborted) {
      assistantMsg.content += '\n\n[Response stopped by user]';
      assistantMsg.streaming = false;
    } else {
      console.error('[ChatService] Error streaming response:', error);
      assistantMsg.content += '\n\n[Error generating response]';
      assistantMsg.streaming = false;
    }
    await this.saveConversation(cardId);
  } finally {
    this.abortControllers.delete(cardId);
  }
}
