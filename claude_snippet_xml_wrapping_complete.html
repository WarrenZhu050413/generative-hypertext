üìñ XML Wrapping Standard

Format Rules

  • Opening tag: First non-empty line must be <tag_name>
  • Closing tag: Last non-empty line must be </tag_name>
  • Tag naming: Use snake_case for multi-word snippet names
  • Content: Everything between tags (including YAML frontmatter if present)
‚úì CORRECT Format: 
<my_snippet>
# Snippet Title

Content goes here...
More content...
</my_snippet>
‚úó INCORRECT Format: 
# Snippet Title

Content goes here...
More content...

(No XML tags!)
